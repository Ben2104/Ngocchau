# Web App

Next.js App Router dashboard for the gold shop management system.

## Responsibilities

- Supabase Auth sign-in and session handling
- TanStack Query data fetching
- Dashboard, operation pages, and import workflow UI
- Thin client-side validation before API submission

## Local Commands

- `pnpm --filter @gold-shop/web dev`
- `pnpm --filter @gold-shop/web build`
- `pnpm --filter @gold-shop/web typecheck`
